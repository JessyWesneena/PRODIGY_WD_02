# Stopwatch Web Application

This project is a Stopwatch Web Application created as part of my internship at Prodigy InfoTech.

# Project Description
The Stopwatch Web Application allows users to start, pause, and reset time. It accurately tracks hours, minutes, and seconds.

#Features
- Start button to begin timing
- Pause button to stop timing
- Reset button to clear time
- Simple and user-friendly interface

# Technologies Used
- HTML
- CSS
- JavaScript
